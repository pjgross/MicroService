During my time as an environments manager I was always left feeling that there must be a better way of architecting our systems, and started looking for techniques to improve my own development at home.


This blog documents what I learned during this development process, documenting my journey into Micro services, documenting what I looked at and then linking to my source code (developed following a course which met all of my requirements (sorry it is in NodeJS & NextJS))

This is the course and most probably one of the best programming courses I have ever done


The source code is almost identical to the courses, although I have made some changes where i did not like his approach and the Kubernetes and Docker files are alot more production like, and I have put as many comments as possible to help the reader (and me in a few years time) understand what the code is doing.


I have placed the code for this course in a Public GitHub repository. https://github.com/pjgross/MicroService.git


Future developments will be done in a private GitLab repository
